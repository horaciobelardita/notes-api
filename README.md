### Sobre

El objetivo era construir una API de notas para ser consumida con front-end con React Js.

### Funcionalidades disponibles

- Listar/registrar/editar/eliminar notas.

### Instalando dependencias

```sh
git clone ...
cd notes-api
npm i
```

### Iniciando el servidor

```sh
npm run dev
```
